# Stack Data Structure Applications (Java)

This project demonstrates fundamental applications of the Stack data structure using Java.

## Implemented Topics
- Infix to Postfix Expression Conversion
- Balanced Brackets Checking
- String / Expression Reversal

## Features
- Custom Stack implementation
- Node-based stack structure
- Console-based input and output
- Executable JAR files for each application

## Technologies
- Java
- Object-Oriented Programming
- Data Structures

## Files
- Stack.java → Stack implementation  
- Node.java → Node structure  
- Main.java → Main program  
- postfix_infix.jar → Infix to Postfix application  
- problembrackets.jar → Balanced brackets checker  
- Reverse.jar → Reverse string application  

## How to Run (Using JAR)
java -jar postfix_infix.jar
java -jar problembrackets.jar
java -jar Reverse.jar


## How to Compile and Run (Source Code)
javac Main.java Stack.java Node.java
java Main
